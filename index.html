THANK YOU!

I hope you enjoyed this cute little story about my dog, Pepper as I did making it! I learned a lot about JavaScript, JQuery, and plugins! One day, I would like to be well-versed in Javascript and JQuery to make my very own plugin-ins to share with the web development world. In the meantime, I owe a big thank you to Pete R’s jQuery One Pace Scroll plugin for helping me achieve that animated scroll I envisioned with the nice navigation bar on the side. Before the project, I was still unsure about JS and how it functioned. After the project, I’m starting to understand a lot more, and I have an idea of what slice of code does what, why, and how. This was exciting! I used what we learned in class to show and hide elements, and I created some fun interactions. Overall, one of the more fun final projects this semester!
